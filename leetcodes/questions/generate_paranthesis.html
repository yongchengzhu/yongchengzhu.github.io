22. Generate Paranthesis

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

For example, given n = 3, a solution set is:

Example:

[ "((()))", "(()())", "(())()", "()(())", "()()()" ]

The most straightforward approach is to use backtracking. First realize what choices we have at the very top of the tree. We have \(n\) left paranthesis and \(n\) right paranthesis available to us. Valid paranthesis cannot start with a right paranthesis, hence when both paranthesis have equal amounts available, there's only one choice to make which is to use the left paranthesis. Next, left paranthesis is now less than right paranthesis, at this state we can make two choices, we can use either left and right paranthesis. Finally, when we run out of left paranthesis, obviously, we only have one choice left which is to use right paranthesis. 

            
              var generateHelper = function(lefts, rights, result, currentParan) {
                  if (!lefts && !rights) {
                      result.push(currentParan);
                      return;
                  }

                  if (lefts === rights) {
                      generateHelper(lefts - 1, rights, result, currentParan.concat('('));
                  } else if (lefts === 0) {
                      generateHelper(lefts, rights - 1, result, currentParan.concat(')'));
                  } else {
                      generateHelper(lefts - 1, rights, result, currentParan.concat('('));
                      generateHelper(lefts, rights - 1, result, currentParan.concat(')'));
                  }
              }

Runtime complexity is too complicated to calculate. The space complexity is \(O(n)\).
