168. Excel Sheet Column Title

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

Example:

Input: 28 Output: "AB"

This problem is a little more difficult than the last. We have to take the base 10 integer and convert it into base 26. However, the number 0 is not mapped to any letters in base 26. To make the math simpler, let's shift the numbers down by 1 (for example, A -> 0, B -> 1, ...). Now, we just need to take the input integer, subtract it by 1 and then find the remainder. Then take the quotient and find the remainder of that, and repeat this process until the quotient reaches 0 and stop. The answer is simply the reverse order of all the remainders we just found. 

            
              var convertToTitle = function(n) {
                  const offset = 'A'.charCodeAt(0);
                  let   remain = (n - 1) % 26;
                  let   result = "";

                  result = String.fromCharCode(remain + offset) + result;
                  n      = Math.floor((n - 1) / 26);

                  while (n !== 0) {
                      remain = (n - 1) % 26;
                      n      = Math.floor((n - 1) / 26);
                      result = String.fromCharCode(remain + offset) + result
                  }

                  return result;
              };

Runtime: \(O(n)\), Space: \(O(1)\).
