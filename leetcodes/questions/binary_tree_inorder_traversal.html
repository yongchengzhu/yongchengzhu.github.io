94. Binary Tree Inorder Traversal

Given a binary tree, return the inorder traversal of its nodes' values.

Example:

Input: [1,null,2,3]
1 \ 2 / 3
Output: [1,3,2]
Follow up: Recursive solution is trivial, could you do it iteratively?

The problem is quite straightforwarad. It's asking for an output of an array containing the tree node values visited by inorder traversal. The most intuitive way to do this is to do a DFS using recursion. 

              
                var inorderTraversal = function(root) {
                    if (!root) {
                        return [];
                    }

                    let left = inorderTraversal(root.left);

                    left.push(root.val);

                    left = left.concat(inorderTraversal(root.right));

                    return left;
                };

The space complexity is obviously \(O(n)\). But, the time complexity is \(O(n^2)\) because we are using concat on the right subtree for each node.
