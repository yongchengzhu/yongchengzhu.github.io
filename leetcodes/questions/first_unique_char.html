387. First Unique Character in a String

Given a string, find the first non-repeating character in it and return it's index. If it doesn't exist, return -1.

Examples:

s = "leetcode" return 0. s = "loveleetcode", return 2.

The brute force solution is my first go-to. However, it's not a good solution and leetcode evaluate this solution as time-limit exceeded. But it's always good to think and talk about the brute force approach in an interview setting. 

            
              var firstUniqChar = function(s) {
                  for (let i = 0; i < s.length; ++i) {
                      let repeated = false;

                      for (let j = 0; j < s.length; ++j) {
                          if (i !== j && s[i] === s[j]) {
                              repeated = true;
                          }
                      }

                      if (!repeated) {
                          return i;
                      }
                  }

                  return -1;
              };

The runtime is \(O(n^2)\) and the space is \(O(1)\).
