136. Single Number

Given a non-empty array of integers, every element appears except for one. Find that single one.

Note: Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?Your algorithm should run in linear runtime complexity. Could you implement it using only constant extra space complexity?

Example 1:

Input: [2, 2, 1]
Output: 1

Example 2:

Input: [4,1,2,1,2]
Output: 4

The most straightforward way is to loop through the input array, and check if each integer is in the hashset. If it doesn't exist in the hashset, then add it inside. Otherwise, remove the existing integer inside the hashset. By the end of the loop, the hashset should have one integer remaining inside, and that's the answer we are looking for. 

            
              var singleNumber = function(nums) {
                  let set = new Set([]);

                  for (let i = 0; i < nums.length; ++i) {
                      if (!set.has(nums[i])) {
                          set.add(nums[i]);
                      } else {
                          set.delete(nums[i]);
                      }
                  }

                  for (let value of set) {
                      return value;
                  }
              };

This algorithm has linear runtime, but \(O(n)\) space complexity.
