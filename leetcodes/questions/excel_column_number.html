171. Excel Sheet Column Number

Given a column title as appear in an Excel sheet, return its corresponding column number.

Example:

Input: "AB" Output: 28

For this problem, I recall how I would convert a binary number to a base 10 number, or numbers from any base to base 10 in general. $$digit_nbase^n + digit_{n-1}base^{n-1} + \cdots + digit_1base^1 + digit_0base^0$$ 

            
              var titleToNumber = function(s) {
                  const offset = 'A'.charCodeAt(0) - 1;
                  const n      = s.length;
                  let   result = 0;

                  for (let i = n - 1, e = 0; i >= 0; --i, ++e) {
                      result += (s[i].charCodeAt(0) - offset) * Math.pow(26, e);
                  }

                  return result;
              };

Runtime: \(O(n)\), Space: \(O(1)\).
