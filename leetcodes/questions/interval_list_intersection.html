986. Interval List Intersections

Given two lists of closed intervals, each list of intervals is pairwise disjoint and in sorted order.

Return the intersection of these two interval lists.

Example:

Input: A = [[0,2],[5,10],[13,23],[24,25]], B = [[1,5],[8,12],[15,24],[25,26]] Output: [[1,2],[5,5],[8,10],[15,23],[24,24],[25,25]] Reminder: The inputs and the desired output are lists of Interval objects, and not arrays or lists.

Approach 1: Two Pointers

The intuitive algorithm I came up with is simple. First, create two pointers, one for each array. Each array contains some number of closed-intervals. Start a loop that runs as long as both pointers are not out of boundary. At each iteration, find out which pointer is pointing to the interval with the greater starting index. For that index, check if it's in-between the interval of the other array. If it is then there is an intersection otherwise there is no intersection. 

            
              var intervalIntersection = function(A, B) {
                  let result = [];
                  let i = 0;
                  let j = 0;

                  while (i < A.length && j < B.length) {
                      if (A[i][0] >= B[j][0]) {
                          if (A[i][0] >= B[j][0] && A[i][0] <= B[j][1]) {
                              A[i][1] <= B[j][1]? (result.push([A[i][0], A[i][1]]), i++):
                                                  (result.push([A[i][0], B[j][1]]), j++);
                          } else j++;
                      } else {
                          if (B[j][0] >= A[i][0] && B[j][0] <= A[i][1]) {
                              A[i][1] <= B[j][1]? (result.push([B[j][0], A[i][1]]), i++):
                                                  (result.push([B[j][0], B[j][1]]), j++);
                          } else i++;
                      }
                  }

                  return result;
              };

Runtime: \(O(n)\), Space: \(O(1)\).
