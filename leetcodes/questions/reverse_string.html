344. Reverse String

Write a function that reverses a string. The input string is given as an array of characters char[].

Do not allocate extra space for another array, you must do this by modifying the input array in-place with O(1) extra memory.

You may assume all the characters consist of printable ascii characters.

Example 1:

Input: ["h","e","l","l","o"]
Output: ["o","l","l","e","h"]

Example 2:

Input: ["H","a","n","n","a","h"]
Output: ["h","a","n","n","a","H"]

The most intuitive solution to me is to do sorting with two pointers; one points to the start of the array and the other points to the end of the array. Then, loop through the array and swap the values at both pointer indexes. The start pointer increments, while the end pointer decrements. The loop terminates when both pointers meet or when they crossed each other. 

        
          var reverseString = function(s) {
              let start = 0;
              let end   = s.length - 1;

              while(true) {
                  if (start >= end) {
                      break;
                  }

                  let temp = s[start];
                  s[start] = s[end];
                  s[end]   = temp;

                  start++;
                  end--;
              }

              return s;
          };

This algorithm satisfies \(O(n)\) runtime and \(O(1)\) extra space.
