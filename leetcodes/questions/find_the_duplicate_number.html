287. Find the Duplicate Number

Given an array nums containing n + 1 integers where each integer is between 1 and n (inclusive), prove that at least one duplicate number must exist. Assume that there is only one duplicate number, find the duplicate one.

Example 1:

Input: [1,3,4,2,2]
Output: 2

Example 2:

Input: [3,1,3,4,2]
Output: 3

Note:

  1. You must not modify the array (assume the array is read only).
  2. You must use only constant, \(O(1)\) extra space.
  3. Your runtime complexity should be less than \(O(n^2)\).
  4. There is only one duplicate number in the array, but it could be repeated more than once.

We can treat the nums array as a linked list with a cycle by representing each value inside nums as a node. Since each value is an integer from 1 to n (inclusive), we can use the value as an index to the next node. Then, we can use Floyd's cycle detection algorithm to find the starting position of the cycle which is the repeated number. 

            
              var findDuplicate = function(nums) {
                  let slow  = 0;
                  let fast  = 0;
                  let start = 0;
                  let meet  = -Infinity;

                  while (true) {
                      slow = nums[slow];
                      fast = nums[nums[fast]];

                      if (nums[slow] === nums[fast]) {
                          meet = slow;
                          break;
                      }
                  }

                  while (true) {
                      if (nums[start] === nums[meet]) {
                          break;
                      }

                      start = nums[start];
                      meet  = nums[meet];
                  }

                  return nums[meet];
              };

This algorithm satisfies all the constraints in the problem.
