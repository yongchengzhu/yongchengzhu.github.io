88. Merge Sorted Array

Given two sorted integer arrays nums1 and nums2, merge nums2 into nums1 as one sorted array.

Note:

  • The number of elements initialized in nums1 and nums2 are m and n respectively.
  • You may assume that nums1 has enough space (size that is greater or equal to m + n) to hold additional elements from nums2.

Example:

Input: nums1 = [1,2,3,0,0,0], m = 3 nums2 = [2,5,6], n = 3 Output: [1,2,2,3,5,6]

Approach 1: Two-pointers

The trick is to initialize two pointers at the largest number in both arrays. Then, compare the numbers at both pointers, copy the larger number over to the end of nums1. Repeat until one of the pointers are out of bound. The corner cases to consider: if \(j\) is still inside nums2, then copy over the rest of nums2 into nums1. If \(i\) is still inside nums1, then do nothing because the rest of the numbers inside nums1 are in the correct places. 

            
              var merge = function(nums1, m, nums2, n) {
                  let end = m + n - 1;
                  let i   = m - 1;
                  let j   = n - 1;

                  while (i >= 0 && j >= 0) {
                      if (nums1[i] > nums2[j]) nums1[end--] = nums1[i--];
                      else                     nums1[end--] = nums2[j--];
                  }

                  while (j >= 0) {
                      nums1[end--] = nums2[j--];
                  }

                  return nums1;
              };

Runtime: \(O(n + m)\), Space: \(O(1)\).
